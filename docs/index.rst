===============
sphinx-revealjs
===============

|THIS| is Sphinx extension
to generate Reveal.js presentation documents
from **standard** reStructuredText.

Basic Features
==============

* Nested sections
* Speaker note
* Syntax highlight for Reveal.js (not used pygments)
* Initialize reveal.js by document
